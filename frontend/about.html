About This Website

We think Python is kind of useless without libraries (unless you're one of those mythical beings who writes neural nets in raw Python).

While PyPI is amazing, it's designed for hosting, not exploring. We wanted something that feels more like a recommendation engine for your next import statement.

That's why we think it is nice to have a website where Python developers can find what fits in their projects.

Most of these libraries were lovingly stolen (legally!) from the legendary awesome-python list, plus a few more sprinkled in for flavor.

This project is open-source, so if you spot a bug, typo, or know a cool library that deserves the spotlight, hop in and contribute! We adore good pull requests almost as much as clean Python syntax.
